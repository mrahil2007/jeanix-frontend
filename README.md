##  Project Overview
JEANIX is a responsive e-commerce website frontend built using **HTML, CSS, and JavaScript**.  
The project focuses on clean UI, mobile-first design, and real-world production deployment.



---

## Features
- Product listing and product detail pages  
- Cart UI with Cash on Delivery (COD) flow  
- Mobile-first responsive design  
- Optimized layout for iPhone and Android devices  
- Clean and minimal user interface  

---

## Tech Stack
- **HTML5**  
- **CSS3**  
- **JavaScript (ES6+)**

---

##  Responsiveness
- Designed using a **mobile-first approach**  
- Fixed UI and overflow issues on iPhone devices  
- Tested across multiple screen sizes
